# Inventario de Casos de Uso - Spinner

## Funcionalidades del Componente

1. **Size** ('xs' | 'sm' | 'md' | 'lg' | 'xl')
   - Tamaño del spinner (default: 'md')

2. **Variant** ('primary' | 'secondary' | 'success' | 'warning' | 'error' | 'info')
   - Variante de color del spinner (default: 'primary')

3. **Animated** (boolean)
   - Si el spinner está animado (default: true)

4. **Label** (string)
   - Texto a mostrar debajo del spinner (opcional)

5. **FullScreen** (boolean)
   - Si el spinner debe ocupar todo el contenedor (centrado) (default: false)

6. **ClassName** (string)
   - Clases CSS adicionales

7. **Style** (string)
   - Estilos inline adicionales
